Why a web version?

Lingua Sacra started on iPhone and Android, where it still works fully offline — on commutes, in the confessional line, during Mass prep in spotty rectory Wi-Fi. But plenty of Catholics who want to learn Latin are at a desk: seminarians studying for morning office, homeschool parents preparing tomorrow's lesson, choir directors marking up hymn translations.

For them, waiting to get home to their phone was the wrong friction. So the full app now runs in any modern browser — same lessons, same Vulgate reader, same spaced-repetition system, same crown levels. A single Supabase account ties it all together: start a lesson at your desk, finish it on your phone on the way to adoration.

Frequently asked

Is this the same app, or a trimmed-down version?

The same app. Same 143 lessons, same Vulgate chapters, same prayers, same crown system, same spaced-repetition review queue. The web build is a React Native Web compile of the iOS/Android codebase, so content parity is automatic.

Does my progress sync?

Yes, as long as you sign in with the same account. Create an account on the web and your XP, streaks, and SRS cards appear on iOS and Android on first login. The reverse works too.

What about offline?

The web app needs a connection. If you want fully-offline Latin — underground parking garages, airplane mode, rectory Wi-Fi outages — install the iOS or Android app. That's what they're built for.

Which browsers are supported?

Any modern evergreen browser: recent Chrome, Safari, Firefox, or Edge. Audio playback uses standard HTML5, so no plugins or extensions are needed.

How do I get a refund?

Web purchases go through Lemon Squeezy, which handles refunds within 14 days for any reason — reply to the purchase confirmation email. Mobile refunds go through the App Store or Google Play per their own policies.
