Lingua Sacra

Privacy Policy

Effective Date: March 12, 2026
Offline-First · Minimal Data

Overview

Lingua Sacra is an educational app for learning Ecclesiastical Latin in the context of the Catholic tradition. Your privacy is important to us. The app is designed to work fully offline, and online features are entirely optional.

Offline Mode (Default)

By default, Lingua Sacra operates entirely on your device. All lesson progress, quiz scores, vocabulary review schedules, and settings are stored locally using on-device storage. In this mode, the app collects no personal data, makes no network requests, and does not communicate with any servers.

Optional Account & Cloud Features

If you choose to create an account, the following data is collected and stored on our servers:

  • Email address — used for authentication and password recovery.
  • Display name — an optional name you choose, shown on group leaderboards.
  • Learning progress — lesson completions, vocabulary mastery, XP, and streak data, synced to enable cloud backup and cross-device access.
  • Group membership — if you join a study group, your display name, XP, streak, and lessons completed are visible to other group members.
  • Daily challenge scores — if you participate in daily challenges, your score and completion time are recorded.

You may use the app without creating an account. All online features are opt-in.

Data Storage & Security

Online data is stored using Supabase, a cloud database service hosted on Amazon Web Services (AWS) in the United States. All data is transmitted over HTTPS and protected by Row Level Security policies, meaning each user can only access their own data (and shared group data for groups they belong to).

Crash Reporting

Lingua Sacra uses Sentry for anonymous crash and error reporting in production builds. Sentry collects technical information about errors (such as stack traces, device model, and operating system version) to help us fix bugs. Sentry does not collect your name, email address, IP address, or any personally identifiable information.

Third-Party Services

Lingua Sacra uses the following third-party services:

  • Supabase (optional, only if you create an account) — authentication and data storage. Supabase Privacy Policy
  • Sentry (production builds only) — anonymous crash reporting. Sentry Privacy Policy

Lingua Sacra does not integrate any advertising, social media tracking, or analytics services.

Children's Privacy

Lingua Sacra is suitable for users of all ages. The app does not knowingly collect personal information from children under 13. If a child creates an account, only an email address and optional display name are collected — the same minimal data collected from all users. Parents or guardians may contact us to request deletion of a child's account.

Data Deletion

You may delete your account and all associated cloud data at any time from within the app. Local data stored on your device can be removed by uninstalling the app. To request data deletion by email, contact us at the address below.
